Accessibility statement

Accessibility statement

This accessibility statement applies to systemc.com

This website is run by System C Healthcare Limited.

We want as many people as possible to be able to use this website. For example, that means you should be able to:

  • zoom in up to 300% without the text spilling off the screen
  • navigate most of the website using just a keyboard
  • navigate most of the website using speech recognition software
  • listen to most of the website using a screen reader (including the most recent versions of JAWS, NVDA and VoiceOver)

We’ve also tried to make the website text as simple as possible to understand.

AbilityNet has advice on making your device easier to use if you have a disability.

How accessible this website is

We know some parts of this website are not fully accessible:

  • All pages have some areas of text on coloured backgrounds, and here the level of contrast between text and background is sub-optimal.
  • Navigation elements and form controls don’t change visibility when they are focused on.
  • Not all images have descriptive alternative text.
  • Some links that overlay images do not have text associated with them to explain what they are via a screen reader.
  • Search and contact form fields are missing label text.
  • On the Our People page, there is no way of navigating between each profile using just the keyboard.
  • On some pages, there are empty headings and empty links within the content.
  • Our downloadable PDF documents are not necessarily fully accessible.

Reporting accessibility problems with this website

We’re always looking to improve the accessibility of this website. If you find any problems not listed on this page or think we’re not meeting accessibility requirements, please contact:

  • enquire@systemc.com
  • 01622 691616

Technical information about this website’s accessibility

System C Healthcare Ltd is not a public sector body but is committed to making its website accessible, in accordance with the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018.

Compliance status

This website is partially compliant with the Web Content Accessibility Guidelines version 2.1 AA standard, due to ‘the non-compliances’ listed below.

Non-accessible content

The content listed below is non-accessible for the following reasons.

Non-compliance with the accessibility regulations

  • All pages have some areas of text on coloured backrounds, and here the level of contrast between text and background is sub-optimal.

We’re planning a redesign of our website, which will address this issue. The new design will be in use by January 2022.

  • Navigation elements and form controls don’t change visibility when they are focused on.

We’re planning a redesign of our website, which will address this issue. The new design will be in use by January 2022.

  • Some images do not have a text alternative, so people using a screen reader cannot access the information.

We plan to add text alternatives for all images by January 2022. When we publish new content, we’ll make sure our use of images meets accessibility standards.

  • Search and contact form fields are missing label text.

We plan to add label text to all forms by January 2022.

  • On the Our People page, there is no way of navigating between each profile using just the keyboard.

We’re planning a redesign of our website, which will address this issue. The new design will be in use by January 2022.

  • On some pages, there are empty headings and empty links within the content.

We plan to remove empty headings and links by January 2022.

Content that’s not within the scope of the accessibility regulations

  • PDFs and other documents

Our downloadable PDF documents are not necessarily fully accessible.

We are currently reviewing the provision of downloadable PDFs on our website and hope to provide this content in either accessible PDF or HTML form by May 2022.

Any new PDFs or Word documents we publish will meet accessibility standards.

What we’re doing to improve accessibility

An accessibility audit of this website was carried out in November 2021. The audit was conducted by our Web Developers, See Green Media Ltd.

Sample pages were tested to cover the different accessibility features. From the results we made an action plan including practical solutions to make this website easier for everyone to use. 

We’re continuing to work to address issues raised in the accessibility audit report, and any future change or redesign of our website will include accessibility as a core requirement.

Preparation of this accessibility statement
